# Project Management TelNet

A simple web application built with **React.js** that helps users **decompose project tasks** and **reorder them**.  
This project was built primarily **for learning purposes**, to practice recursion, problem-solving, and drag & drop functionality in a real-world app.

---

## Features

This app allows you to:

- Authenticate using JWT with **asymmetric encryption**
- Create projects
- Create project tasks with **drag & drop functionality**
- Invite friends to implement tasks
- Check whether two tasks will form a **cycle** (to prevent uncompletable projects)
